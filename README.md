# Chat
Este proyecto es una aplicación de chat en tiempo real desarrollada con Node.js, Express y Socket.IO, cuyo objetivo es entender cómo funciona la comunicación bidireccional entre cliente y servidor usando WebSockets.

El chat permite que varios usuarios envíen y reciban mensajes de forma instantánea a través del navegador, sin necesidad de recargar la página. Además, se implementó una estructura básica de archivos y estilos con HTML y CSS, lo que facilita su comprensión y futura expansión.

Este proyecto está pensado como una práctica introductoria para aprender conceptos clave como:

Servidores con Node.js

Uso de Express para servir archivos estáticos

Comunicación en tiempo real con Socket.IO

Organización básica de un proyecto web

Ideal como base para seguir agregando funcionalidades como usuarios, salas de chat o persistencia de mensajes.
